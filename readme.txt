== Description ==

An enhancement for Visual Composer to replace prettyPhoto lightbox support to support for any lightbox.

== Installation ==

Installing "Visual Composer Lightbox" can be done either by searching for "Visual Composer Lightbox" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
